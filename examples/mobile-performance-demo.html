Mobile Performance Demo

Test different performance modes while panning around the map on mobile devices

Performance Modes:

Smooth Mode (Default): Tiles load continuously while panning for the smoothest experience. Uses more bandwidth but provides better UX.

Performance Mode: More conservative tile loading - better for slower devices or limited bandwidth. Tiles load after panning stops.

Try panning around the map in different modes to see how tile loading behavior changes. On mobile devices, the smooth mode will prevent empty tiles during panning.
